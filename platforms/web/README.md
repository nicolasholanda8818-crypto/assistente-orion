# ORION Web

## Papel

Interface compartilhada do ORION em navegador e PWA.

## Baseline

- manifest;
- service worker;
- offline shell;
- REST e WebSocket locais;
- sem segredo persistente no navegador.

## Gate Para Acesso Remoto

HTTPS, WSS, identidade, pareamento, validacao de origem e limites devem existir antes
de conectar um navegador fora do host.
