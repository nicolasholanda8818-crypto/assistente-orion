# ORION macOS

## Papel

Host desktop planejado para executar Python, FastAPI, SQLite e recursos locais.

## Adapters Futuros

- `~/Library/Application Support/Orion`;
- notificacoes desktop;
- Keychain para protecao de chave;
- metricas do sistema;
- controle de programas por allowlist.

macOS tambem e o ambiente necessario para build e assinatura do cliente iOS.
