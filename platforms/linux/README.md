# ORION Linux

## Papel

Host desktop planejado para executar Python, FastAPI, SQLite e recursos locais.

## Adapters Futuros

- diretorios XDG;
- notificacoes desktop;
- metricas do sistema;
- cofre de chaves suportado pelo ambiente;
- controle de programas por allowlist.

Nenhuma operacao do host entra no Core sem Tool System, permissao e auditoria.
