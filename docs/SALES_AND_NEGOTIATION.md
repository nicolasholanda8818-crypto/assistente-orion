# Orion Vendas e Negociacao

Data: 2026-06-24

## Objetivo

Adicionar uma base inicial para o Orion orientar vendas, atendimento, propostas, follow-up e negociacao de forma etica.

## Intencoes

- `sales`: usuario quer vender um servico ou produto.
- `sales.script`: usuario quer roteiro de venda.
- `sales.message`: usuario quer mensagem profissional para cliente.
- `negotiation`: usuario precisa negociar.
- `objection.price`: cliente disse que esta caro.
- `consultant.senior`: usuario pede visao de consultor experiente.

## Padrao de Resposta

O Orion prioriza:

- descoberta da necessidade;
- proposta de valor;
- pergunta inteligente;
- postura profissional;
- proximo passo claro;
- alternativas antes de desconto.

## Limite Seguro

O Orion nao promete resultado comercial garantido e nao manipula clientes. A orientacao deve ser pratica, profissional e etica.
